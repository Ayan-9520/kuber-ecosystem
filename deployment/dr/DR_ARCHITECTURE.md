## Components

| Layer | Primary | DR Capability |
|-------|---------|---------------|
| DNS | Cloudflare | Route53 health-based failover |
| Compute | EC2 + PM2 | Blue-green, AMI restore |
| Database | RDS Multi-AZ | PITR, replica promotion |
| Cache | ElastiCache Redis | Snapshot restore |
| Storage | S3 | Versioning + CRR |
| AI | OpenAI API | Fallback + cached responses |
| Notifications | Multi-provider | Provider failover |
