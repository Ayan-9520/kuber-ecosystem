# Production Launch Checklist

## T-14

- [ ] TestFlight external beta sign-off
- [ ] Store listing finalized (both apps)
- [ ] Marketing assets ready

## T-7 — Management Team Briefings

- [ ] **Branch Managers** — branch rollout, partner comms, SLA monitoring
- [ ] **Regional Managers** — regional targets, escalation paths
- [ ] **Sales Team** — customer app flows, AI Advisor, lead capture
- [ ] **Credit Team** — loan application accuracy, credit disclaimers
- [ ] **Operations Team** — document upload, OCR, lender handoff
- [ ] **Compliance Team** — privacy labels, consent, financial disclaimers
- [ ] **Support Team** — in-app support, review responses, Voice AI escalations
- [ ] **Admin Team** — CRM dashboards, release runbooks, incident contacts

## T-7 — Technical

- [ ] Production IPA uploaded
- [ ] Phased release enabled (7-day default)
- [ ] Monitoring dashboards ready

## Launch

- [ ] Submit for App Review
- [ ] Monitor review status (24–48h typical)
- [ ] On approval: release with phased rollout
- [ ] Record in CRM App Store dashboard

## Rollback

1. Remove app from sale (extreme) or halt phased release
2. Submit expedited hotfix with new build number
3. Document in production/incident module

## Post-Launch (48h)

- [ ] Crash-free sessions ≥ 99.5%
- [ ] Review responses within 48h
- [ ] Launch report generated
