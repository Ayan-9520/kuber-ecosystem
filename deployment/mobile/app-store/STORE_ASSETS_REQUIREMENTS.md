# App Store Assets Requirements

| Asset | Spec | Customer | DSA |
|-------|------|----------|-----|
| App Icon | 1024×1024 PNG, no alpha | ✓ | ✓ |
| iPhone 6.7" Screenshots | 1290×2796 or 1284×2778 | Min 3 | Min 3 |
| iPhone 6.5" Screenshots | 1242×2688 or 1284×2778 | Min 3 | Min 3 |
| iPhone 5.5" Screenshots | 1242×2208 | Optional | Optional |
| iPad Pro 12.9" | 2048×2732 | Min 1 | Min 1 |
| App Preview Video | 15–30s, H.264 | Optional | Optional |
| Promotional Text | 170 chars, editable without review | ✓ | ✓ |
| Marketing URL | https://kuberone.com | ✓ | ✓ |

Source icons: `apps/mobile-*/assets/icon.png`

Export path: `deployment/mobile/app-store/assets/{customer|dsa}/`
